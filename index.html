My Semantic HTML Cheat Sheet

Tag Description
<section> According to W3C's HTML documentation: "A section is a thematic grouping of content, typically with a heading."
<article> This element specifies independent, self-contained content
<header> This element represents a container for introductory content or a set of navigational links
<footer> This element defines a footer for a document or section
<nav> This element defines a set of navigation links
<aside> The content should be indirectly related to the surrounding content
<fig> This tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc
<figcaption> This tag defines a caption for a figure element
